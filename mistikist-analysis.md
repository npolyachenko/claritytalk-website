# Mistikist Website Analysis

## Overview
Mistikist.com is a **WordPress website** built with **Elementor** page builder. It's a landing page for a neuroscience-based mental wellbeing mobile app.

## Technical Stack

### Platform & Builder
- **CMS**: WordPress 6.8.3
- **Theme**: Astra Theme (v4.11.14) - A lightweight, highly customizable WordPress theme
- **Page Builder**: Elementor Pro - Drag-and-drop page builder
- **Plugins Detected**:
  - Yoast SEO (v26.3)
  - Google Tag Manager for WordPress (gtm4wp)
  - Cookie Law Info (v3.3.6)
  - Astra Portfolio
  
### Design System
- **Fonts**: 
  - Poppins (200, 400 weights)
  - Raleway (600 weight)  
  - Roboto (100, 300 weights)
- **Colors**: Dark theme with white text (#FFFFFF) on dark backgrounds (#000000, #1a052b)
- **Button Style**: Outlined buttons with rounded corners (100px border-radius)

## Key Features & Patterns

### Visual Effects
- **Scroll animations**: fade-in effects on cards and sections
- **Motion effects**: parallax-style scrolling on images
- **Animated content**: staggered animations with delays

### Content Strategy
- Heavy use of **scientific credibility** (research citations, stats)
- **App store presence** emphasized (download buttons, ratings)
- **Trust signals**: supporter logos, user count, testimonials
- **Educational approach**: explains the science behind the product

### Call-to-Actions
- Multiple "Get it now!" buttons throughout
- Links to `/premium` page
- App store badges (Google Play, App Store)

## Media Assets
All hosted on WordPress uploads directory:
- **Images**: WebP format for optimization
- **Responsive images**: Multiple sizes with srcset
- **SVG icons**: Used for simple graphics
- **Video**: YouTube embed (promotional video)

## Mobile Optimization
- Hamburger menu for mobile navigation
- Responsive breakpoints at 921px and 544px
- Touch-friendly button sizes
- Stack layout on mobile devices

## Performance Features
- Lazy loading on images
- CDN usage (fonts.googleapis.com, cdnjs.cloudflare.com)
- Minified CSS
- Deferred JavaScript loading
